## How to Compile and Run
Run the following command from the project root directory:

Compile:
javac -cp lib/stdlib.jar:lib/algs4.jar -d bin/ src/*.java

Run:
java -cp ./bin:lib/stdlib.jar:lib/algs4.jar Khoe noGUI <input_file> <handin> <search_type>
 - <input_file> = path to input file
 - <handin> = 1 (midterm) or 2 (final)
 - <search_type> = required only for handin 2

Search types:
 - w = word
 - e = definition
 - d = dialect
 - o = origin
 - p = part of speech
